# Приложение-шаблон для ресторанов

## Запуск

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Запустите приложение в режиме разработки:

   ```bash
   npm run dev
   ```

3. Сборка продакшен-версии:

   ```bash
   npm run build
   ```

4. Предпросмотр собранного приложения:

   ```bash
   npm run preview
   ```

Приложение основано на Vite + React + TypeScript.
